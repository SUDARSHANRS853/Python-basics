1. ## Largest of a given number
```
a = 23
b = 4
c = 5
d= 28
e= 349
largest = max(a, b, c,d,e)
print("Largest number is:", largest)
```
```
num = [23, 4, 5, 28, 349]
maximum = num[0]
for n in num:
    if maximum<n:
        maximum=n
print('Largest number is:',maximum)
```
Output
```
Largest number is:349 
```
